# CALL-CENTER-PERFORMANCE-DASHBOARD-January-March-2021-
This dataset captures operational and customer service metrics from a call center over a three-month period (January–March 2021). It is structured to provide a comprehensive view of performance across agents, topics, and time, making it suitable for analysis, visualization, and reporting in business intelligence contexts.

---

## 📂 Project Overview
The dashboard helps answer:
- Workload Trends: How many calls are received each month, and are there seasonal fluctuations?
- Service Quality: What is the resolution rate and average answer speed, and how do these impact customer satisfaction?
- Agent Performance: Which agents consistently achieve higher satisfaction scores, and where is improvement needed?
- Topic Distribution: What types of issues (streaming, payment, technical support, etc.) generate the most calls?
- Customer Experience: How satisfaction levels vary across topics and agents over time.

---

## 🔐 Key Metrics (Overall)
 High-Level KPIs:

## 🧹 Data Preparation Steps 
- Data Collection 
- Data Cleaning – Remove duplicates, handle missing values, and standardize formats
- Data Transformation : Convert timestamps into monthly and daily aggregates.
- Categorize calls by topic (Streaming, Payment, Technical Support, etc.).
- Built DAX measures for : answered rate, and average answer speed.
- Feature Engineering – Create derived metrics (e.g., satisfaction by agent, calls per topic percentage, monthly call volume).
- Validation – Cross-check totals and averages against system reports to ensure accuracy.
- Visualization Preparation
